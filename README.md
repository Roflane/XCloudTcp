### XCloudTcp
Implementation of client-server communication.

#### Features - Command
1. Register - Register
2. Auth - Auth
3. View Root Directories - dvr
4. Create Directory - dc
5. Delete Directory - dd
6. Rename Directory - dr
7. Download File - fd
8. Upload File - fu
9. Delete File - frm
10. Rename File - fr
11. Copy File - fc
